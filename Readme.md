## Mern stack

============================================================================

### Front end

#### Install

```sh
    yarn
```

#### Download schema json

```sh
    yarn schema:download
```

#### Generate code graphql

```sh
    yarn schema:generate-watch
```

#### Run

```sh
    yarn start
```

============================================================================

### Back end

#### Run docker compose

```sh
    cd docker
    docker-compose up
```

#### Run be

```sh
    yarn start:dev
```
